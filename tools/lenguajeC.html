¿Por qué elegí lenguaje C?

  • Es un lenguaje con pocos requerimientos externos.
  • Tiene pocas palabras reservadas que memorizar.
  • Trabaja muy cerca del lenguaje máquina.

Palabras reservadas:

auto do goto short typeof break double if signed typeof_unqual case else int sizeof union char enum long static unsigned const extern main struct void continue float register switch volatile default for return typedef while

Función main

La función main es la que se ejecutará cuando inicie el programa por lo que ésta debe ser declarada.

int main(){}
